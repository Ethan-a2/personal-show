# Markdown 简历展示

这是一个参考 CodeCV 高级前端工程师模板制作的纯正文静态简历展示：`resume.md` 负责内容编辑，`index.html`、`styles.css` 和 `script.js` 负责版式和打印。

## 使用方式

1. 编辑 `resume.md` 顶部 frontmatter 和正文内容。
2. 在当前目录启动静态服务：

   ```bash
   python3 -m http.server 8000
   ```

3. 打开 `http://localhost:8000` 预览。
4. 使用浏览器打印功能即可导出 PDF。

## 内容约定

- `##` 表示简历一级模块，会渲染成带横线的紫色标题。
- `### 名称 | 时间` 表示经历条目，会自动左右分栏展示名称和时间。
- 普通 `-` 列表用于职责、技能、项目成果。
- 反引号内容如 `` `985` `` 会渲染成标签。

## 方案说明

- 推荐把「可编辑内容」放在 Markdown，把「复杂排版」留给 HTML/CSS。
- 当前实现无构建依赖，适合直接托管到 GitHub Pages、Nginx 或对象存储。
- 如果后续需要离线单文件交付，可以再加一个构建脚本，把 `resume.md` 注入成纯静态 HTML。
